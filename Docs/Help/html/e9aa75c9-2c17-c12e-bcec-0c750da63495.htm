watchasayin.Kryptonite Namespace
Send Feedback
The namespace of the Kryptonite module. The Kryptonite module is resonsible for providing various cryptographic services to the Surface module, e.g. doing an AKE.

Classes

  ClassDescription
ExceptionCryptoFailed
Exception is thrown when a severe error occurs during the connection-selection phase.
SharedLibrary
Class implementing the cryptographic functionality required by Kryptonite by loading an unspecified external library as In-Process COM server and using functionality supplied by that library over COM.

Interfaces

  InterfaceDescription
IKryptonite
Interface to the Kryptonite module. Each Kryptonite module is responsible for accomplishing the following tasks: 1. Provide the designated Surface module with cryptographic services. - Secure authenticated key-exchange (AKE) with a remote Kryptonite module. - Message en/decryption. - Assurance of freshness of messages. - Assurance of authenticity of messages. 2. Enable the user to manage their private keys. E.g. providing a GUI. 3. Enable the user to validate public keys/fingerprints of communication partners.
IKryptoniteExternalCommunication
Extends the IKryptonite interface with functionality to receive and send Kryptonite messages via an external provider.
IKryptoniteExternalCommunicationSpecific
A session bound specific version of IKryptoniteExternalCommunication.
IUserIO
Definition of a user / Kryptonite interaction module.
